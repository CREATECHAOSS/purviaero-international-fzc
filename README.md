# Purvi Aero International FZC

Modern, professional, and highly responsive website for **Purvi Aero International FZC**.

## Features
-   **Precision Aviation Spares**: Quality-assured parts with full traceability.
-   **24/7 AOG Desk**: Rapid response to minimize aircraft downtime.
-   **Global Sourcing**: Extensive network for sourcing expendables, consumables, and rotables.
-   **Modern UI**: Built with Next.js, Tailwind CSS, and optimized for all devices.

## Tech Stack
-   **Frontend**: Next.js, React
-   **Styling**: Tailwind CSS
-   **Fonts**: Inter, Outfit
